# The labs

This book develops the physics of glaciers using a series of self-contained lab exercises that you run yourself, change, and learn from by experiment. The labs are distributed through the book rather than collected at the end. Each points to the chapter where they are introduced for the theory, so the labs and the main text are read together. The lighter labs need only the standard scientific Python stack numpy and matplotlib, and reproduce an analytical result or plot a relationship from the text, such as Glen's creep curve, the Vialov profile, or the effective pressure in a channel against a cavity. The heavier labs build a working glacier model with [icepack](https://icepack.github.io/), which runs on the finite-element library [Firedrake](https://www.firedrakeproject.org/), and these solve the diagnostic and prognostic flow problems on real geometries. A few labs use other group tools, such as the ApRES processing library or the specfab fabric model.

All of the labs are written as Jupyter notebooks and are meant to be run in the book's Docker image, which provides icepack, Firedrake, and the other dependencies already installed, with a kernel named "Firedrake (icepack)". The setup is shown in {doc}`running-icepack-docker`. Because that stack is too heavy to run on a documentation server, the labs are not executed when the book is built. To see the output and do the exercises you must run the notebook yourself in the container.
